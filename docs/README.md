## Getting Started

### Prerequisites

- Rust 1.92+
- Julia 1.12+
- Node.js 18+ (TypeScript)
- PowerShell 7+
- Python 3.9+ (optional, for Maigret)

### Installation

```bash
# Clone repository
git clone https://github.com/Rigohl/JARVIX-MULTISTACK.git
cd JARVIX-MULTISTACK

# Build Rust engine
cd engine && cargo build --release
cd ..

# Install Julia dependencies
julia -e 'using Pkg; Pkg.add("JSON"); Pkg.add("DataFrames")'

# Install Node dependencies (optional)
cd app && npm install && cd ..
```

### First Run

```bash
# Test the CLI
./engine/target/release/jarvix --help

# Run with sample data
./engine/target/release/jarvix collect --run test_001 --input data/seeds.txt
```
